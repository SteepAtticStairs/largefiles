This page is meant to serve as a guide for the way I name my files. There is a specific way how files are named, which is pertinent to what game / video they are. This aims to be a complete, comprehensive guide.

Each file name generally has a number to the left of it, the name of the video (this can be the team that Kentucky is playing, the description of the video, things like that), the date that the video was aired, and a filename extension.

        All of these parts are not on all of the files, because not all of these descriptors are necessary for all videos. I will walk through each possible file name structure, give examples, and explain how they are named.

1. Full Game Replays

        These videos are full game replays from the regular season. This is an example of how these files are named:

                21Kansas_1.29.2022.mp4

I have divided this into several segments, so I can explain the individual parts easier.

First, the 21 part is the game number. This number just means what game of the season Kentucky played this team. Since the file above was named 21Kansas, that means that the game was played as the 21st game of the season.

Second, the Kansas part is a description of the video. The description is almost always the team that Kentucky plays, except for preseason and extra videos. There are some instances, like in the SEC Tournament or the NCAA Tournament, that the description adds additional information about the game played. For example, you might have a game like 39Louisville_NCAATournamentFinalFour_3.31.2012, where the description is highlighted in blue. The NCAATournamentFinalFour is just to tell you that the game was the Final Four in the NCAA Tournament.

Third, and very briefly, the _ part is just an underscore separator for different parts of the file name, so they don’t get jumbled together.

Fourth, the 1.29.2022 part is the date. This is very simple, it is just when the game was played, or when the video was aired. It is organized in month.day.year format, so 1.29.2022 would be January 29, 2022.

Finally, the .mp4 is just the filename extension. You might have more of these than just .mp4, but this is also pretty self-explanatory.

2. Preseason / Exhibition Games

        These videos are events that happen before the regular season, such as Big Blue Madness, the Blue-White Game, and exhibition games. For example, the Big Blue Madness file for 2021-22 would be named as such:

                -00BigBlueMadness_10.14.2011.mp4

        This is the same naming system as the full game replays, but the thing different is the game number. Since preseason videos are not games of the season, they are marked as 00. The dash at the start is to ensure that they are listed first when putting the files for a season in alphabetical order.

        For exhibition games, they are very similar to preseason games, but the game number is changed a little bit. For example:

                -02Asbury_11.6.2016.mp4

        This video is the second exhibition game of the season, played against Asbury, on November 6, 2016. The game number is 02 now, which is different from what we have seen before. The second number, a 2, is to tell you that it is the second exhibition game of the season. The first number, a 0, is to tell you that it is not a regular season game, but an exhibition game. Again, the dash at the start is to ensure that these files are listed first when putting the files for a season in alphabetical order.

3. Extra Videos

        These videos can range anywhere from game highlights to trophy presentations. They are videos that are relevant to that basketball season, but they are not generally straight game replays. They are listed at the end of the file list for a basketball season. For example, the video for the pregame of the 2012 Championship is named:

                z_NationalChampionshipFULLPregame

This is a little different from other file names, but it follows the same rules.

First, there is no date on the game, because it can either be figured out easily by the user, or there is no date to put on the video. For example. z_2011-12KentuckyTournamentHighlights could not have a date, because highlights are compilations which don’t air on a certain day.

Second, the game number has been replaced by a z_. The z underscore is for two reasons. First, there is no game number for these videos, because they are not full game replays. The reason they are a z is to ensure that they are listed last when putting the files for a season in alphabetical order. The underscore is so that the z doesn’t get confused and added to the description part of the video’s file name.

There are descriptions of their own on some of the season’s webpages, so those might help you out as well.
